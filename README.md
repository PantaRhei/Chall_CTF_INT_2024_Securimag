# Chall_CTF_INT_2024_Securimag

**Thème : Hardware / SDR**
Défis créés pour le CTF d’intégration 2024 organisé par **Sécurimag**.

## Challenge 1 – Communication Arduino

Deux microcontrôleurs Arduino communiquent par radio via la librairie **RadioHead**.
Cette communication a été interceptée à l’aide d’un **RTL-SDR**.
L’objectif est d’analyser le signal capturé et d’en **extraire les données échangées**.

## Challenge 2 – Audio FM

Un fichier audio modulé en **FM** est fourni.
Après démodulation, le **flag** est accessible directement dans le contenu audio.

# Chall_CTF_INT_2024_Securimag (EN)

**Theme: Hardware / SDR**

## Challenge 1 – Arduino Communication

Two Arduino microcontrollers communicate wirelessly using the **RadioHead** library.
